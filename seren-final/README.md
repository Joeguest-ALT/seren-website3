# Seren Support Services Website

The official website for Seren Support Services — a family-owned social care provider in South Wales since 2011.

## Structure

```
.
├── index.html           Homepage
├── about.html           Our Story
├── why-seren.html       Why Seren
├── team.html            Meet the Team
├── awards.html          Awards
├── where-we-work.html   Where We Work
├── services.html        Our Services
├── join-us.html         Careers
├── contact.html         Contact
└── assets/              Images and logos
```

## Development

This is a static HTML website. To preview locally, just open `index.html` in your browser, or serve the folder with any static file server.

## Deployment

Hosted on Vercel — deployments are automatic on push to the main branch.
